# Figma Integration

This system is mirrored in Figma using Variables.

---

## Mapping

- Core tokens map to primitive variables
- Semantic tokens map to semantic variables
- Components reference semantic variables only

---

## Naming

Variable names match token paths to keep parity between design and code.

---

## Source of truth

This repository is the single source of truth, while Figma files mirror and derive from it.
