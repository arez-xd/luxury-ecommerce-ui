# Data Visualisation

## Role

The system is not data-heavy, but it still needs a dependable palette for lightweight reporting surfaces such as order states, saved-item metrics, campaign summaries, and account insights.

## Token source

Use:

- `color.data.series.1`
- `color.data.series.2`
- `color.data.series.3`
- `color.data.series.4`
- `color.data.series.5`
- `color.data.series.6`

## Rules

- Reserve gold for the primary or highlighted series.
- Avoid using status colors as chart categories unless the chart truly represents status.
- Pair chart fills with neutral typography and subtle grid or divider strokes.
- Never introduce untracked chart colors in product code.
